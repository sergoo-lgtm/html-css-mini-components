# 👟 Adidas Responsive Landing Page

A sleek and modern product landing page for Adidas running shoes. This project demonstrates responsive web design using pure HTML and CSS, featuring creative background shapes and smooth flexbox layouts.

## ✨ Features
- **Fully Responsive:** Adapts beautifully to both desktop and mobile screens using CSS Media Queries.
- **Modern UI:** Uses CSS `clip-path` for dynamic background styling.
- **Flexbox Architecture:** Clean and centered alignment of all UI elements.
- **Glass-like Container:** Semi-transparent main container for a premium product feel.

## 🛠️ Technologies Used
- Semantic **HTML5**
- **CSS3** (Flexbox, Media Queries, Clip-path, Hover Transitions)

## 🚀 How to Run
1. Open the project folder.
2. Open `index.html` in your web browser to view the page.
3. Resize the browser window to see the mobile responsive layout in action!
