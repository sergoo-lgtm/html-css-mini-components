# 🍽️ Animated Menu

Welcome to the **Animated Menu** project!  
This project is a clean and interactive restaurant menu section built using **HTML5 and CSS3**.  
It features a 4-column grid layout with advanced, sequenced CSS animations that reveal item details when users hover over the menu cards.

## ✨ Features

- **Interactive Hover Effects:** Smooth, multi-step animations using CSS transitions and delays.
- **Sequenced Text Entrance:** The caption text slides in from the side (`translateX`) after the initial overlay animation completes.
- **Dynamic Image Scaling:** Menu images smoothly zoom in (`scale`) when the card is hovered.
- **Spinning Overlay:** A dark, semi-transparent overlay rotates into view (`rotate(360deg)`) while fading in.
- **Clean Float Layout:** A structured 4-column grid built using CSS floats.

## 🛠️ Technologies Used

- **HTML5** – Page structure and element nesting
- **CSS3** – Styling, layout, and visual aesthetics
- **CSS Transforms** – `scale`, `rotate`, and `translateX` for dynamic movement
- **CSS Transitions** – Timing and delays to sequence the animations seamlessly
- **Float Layout** – Used to create the 4-column item grid

## 🚀 How to Run

1. Clone this repository or download the files.
2. Make sure the **image** folder contains your food images (e.g., `rigel-food-sec-2-img1-200x200.jpg`).
3. Open `index.html` in your web browser to see the animations in action.

## 🎯 What I Practiced in This Project

- Advanced **CSS Transforms** (`translate`, `rotate`, `scale`).
- Sequencing animations using **transition delays** (`transition: transform 1s 2s`).
- Absolute and relative positioning for hover overlays.
- Managing classic CSS layouts using **floats** and clearfixes (`clear: both`).
- Building clean, professional UI components.
