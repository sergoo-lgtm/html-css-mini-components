## ✨ Features
* **Fully Responsive:** Adapts seamlessly to both desktop and mobile screens.
* **Modern UI:** Clean, attractive layout capturing Pepsi's signature brand identity.
* **Smooth Animations:** Interactive hover effects on the product images and buttons for a dynamic user experience.
* **Flexbox Layout:** Efficient and flexible component structuring.

## 🛠️ Technologies Used
* **HTML5** for the structure.
* **CSS3** (Flexbox, Media Queries, Transitions) for styling and responsiveness.

## 🚀 How to Run
1. Clone this repository or download the project files.
2. Ensure all image assets (`pepsi_logo.png`, `pepsi_can.png`, `three_pepsi.png`) are located in the same root directory.
3. Open the `index.html` file in your favorite web browser to view the project.
