# 📰 Modern Blog Cards

Welcome to the **Modern Blog Cards** project!  
This project is a clean and simple blog section layout built using **HTML5 and CSS3**.  
It displays blog posts in a card-style layout with images, dates, titles, descriptions, and a "Read More" link.

## ✨ Features

- **Clean Blog Layout:** Simple and modern blog card design.
- **Hover Effects:** Smooth hover animation on the main heading.
- **Card Structure:** Each blog post is displayed inside a styled card.
- **Image + Content Layout:** Blog image with text content inside a neat container.
- **Reusable Components:** The card layout can be reused for blogs, products, or articles.
- **CSS Pseudo Elements:** Decorative lines under the main heading using `::before` and `::after`.

## 🛠️ Technologies Used

- **HTML5** – Page structure
- **CSS3** – Styling and layout
- **Float Layout** – Used to create the 3-column blog grid
- **CSS Transitions** – For smooth hover effects

## 🚀 How to Run

1. Clone this repository or download the files.
2. Make sure the **images folder** contains all required images.
3. Open `index.html` in your browser.
